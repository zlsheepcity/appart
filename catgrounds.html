Catgrounds

Short description:

Update: 2019.8.4

An abstract shapes
as background-images
for visual attraction.

· Three shapes available

· Use CSS code

Long description & examples:

For developers without design skills.

Plain background avoiding, — it is a common trick in design to make visual picture more interesting and attractive. Any abstract form in the back works as a quick solution for the trick.

The Cats. They are just watching. Good for background.

How to use:

1. Copy CSS code into your project. Each catground classname is independent and ready to use.

2. Use CSS classname to activate catground on element
